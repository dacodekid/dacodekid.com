# Factorial Combination

For a given `Total Number People` and `Team Size`, find the total combinations `Teams` you can generate

## Example

For a group of `4 people` and a team size of `2`, the output should be `6`.

If their names are `A`, `B`, `C`, `D` then the combinations are

- `AB`
- `AC`
- `AD`
- `BC`
- `BD`
- `CD`
  - **(Total of 6)**
