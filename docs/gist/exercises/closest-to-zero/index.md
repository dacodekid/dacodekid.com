# Closest To Zero

For a given numbers, find the closest number to 0. If a negative & positve numbers are same, print the positive number.

## Example

For `[5, -5, 2, 0, 7, 8]`, the closest number is `2`.
For `[5, -5, 1, 0, -1, 4]`, the closest number is `1`.
