# Delete Digit

Given some integer, find the maximal number you can obtain by deleting exactly one digit of the given number.

## Example

- For `n = 152`, the output should be

  ```:no-line-numbers
  delete_digit(n) = 52
  ```

- For `n = 1001`, the output should be

  ```:no-line-numbers
  delete_digit(n) = 101
  ```

- **\[input\] integer n**
